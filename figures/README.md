# Figures

This directory contains framework visualizations and result figures used in the README.

| File | Description |
|------|-------------|
| `logo.svg` | Project logo — brain icon with anomaly highlight |
| `architecture.svg` | Full framework architecture diagram (Encoder → Latent → Decoder + skip connections) |
| `pelvic_roc.png` | ROC curve from Pelvic MRI experiment *(generated after training)* |
| `brain_roc.png` | ROC curve from Brain MRI experiment *(generated after training)* |
| `anomaly_map_example.png` | Example input / reconstruction / anomaly-map triplet *(generated after training)* |

> Figures marked *generated after training* will appear here once you run `evaluate.py`
> with the `--visualize` flag.
